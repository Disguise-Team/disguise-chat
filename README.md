# Disguise Chat

## Privacy by Design. Communication by Choice.

Disguise Chat is a privacy-first messaging application designed to give users greater control over how their conversations appear in everyday life.

Unlike traditional messaging apps, Disguise Chat opens as a fully functional calculator. Entering a Secret PIN unlocks a hidden messaging interface, adding an extra layer of discretion before a conversation even begins.

---

## Why Privacy Is Evolving

Modern messaging apps have made encryption widely available, but privacy today extends beyond protecting messages in transit.

People increasingly want control over how their conversations appear on their devices, especially in shared environments, workplaces, or public spaces.

Privacy is becoming an experience—not just a security feature.

---

## Core Features

- 📱 Fully functional calculator interface
- 🔐 Secret PIN to access hidden chats
- 👤 Anonymous accounts without phone number or email
- 🔒 End-to-end encrypted messaging
- ⏱️ Automatic relocking after inactivity
- ⚡ Panic Code to quickly remove local conversations

---

## Privacy Philosophy

Disguise Chat is built around one simple idea:

> **Privacy begins before encryption.**

By combining interface privacy with secure communication, the app helps users manage both message protection and message visibility.
